## How to Create a New React Project with Vite?

Using Vite is one of the recommended way to build react project

1. `npm create vite@latest`
   - It will ask to enter the project name
   - framework
   - varient
2. Move to the new project folder with `cd new-project`
3. Then run `npm install`
4. To run a server use `npm run dev`
